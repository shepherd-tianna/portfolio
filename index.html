Tianna-Bot (Coming Soon)

In this project, I am building a chatbot using the ChatGPT API. ChatGPT is a language model trained by OpenAI that can generate human-like responses to text-based inputs. The chatbot will be designed to respond to user inputs and engage in natural language conversations. The project will involve several steps, including setting up a development environment, integrating the ChatGPT API into the codebase, designing the chatbot's conversation flow, and training the chatbot to respond to user inputs. The end result will be a functional chatbot that can carry out conversations with users, providing information and responding to queries. This chatbot could be used in a variety of applications, such as customer support, personal assistants, and virtual assistants. I've conveniently named the chatbot 'Tianna' because my girlfriend often asks me questions instead of Googling. Hopefully 'Tianna-bot' gets the answer right more than I!
